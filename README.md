# Titanic Dataset Analysis - Task 2 🚢

### Project Overview
[span_0](start_span)[span_1](start_span)This project is part of my **Data Science & Analysis Internship** at **Maincrafts Technology**[span_0](end_span)[span_1](end_span). [span_2](start_span)[span_3](start_span)The goal of Task 2 is to perform Exploratory Data Analysis (EDA) on the famous Titanic dataset to identify factors that influenced the survival of passengers[span_2](end_span)[span_3](end_span).

### Task Objectives
* **[span_4](start_span)[span_5](start_span)Data Loading:** Loaded the Titanic dataset containing passenger details like age, class, and gender[span_4](end_span)[span_5](end_span).
* **[span_6](start_span)Data Cleaning:** Handled missing values, specifically for the 'Age' column, using median imputation[span_6](end_span).
* **[span_7](start_span)Exploratory Analysis:** Analyzed survival patterns based on Gender, Passenger Class, and Age Groups[span_7](end_span).
* **[span_8](start_span)[span_9](start_span)Data Visualization:** Created meaningful plots using Seaborn and Matplotlib to represent the findings[span_8](end_span)[span_9](end_span).

### Key Insights
* **Gender:** Females had a significantly higher survival rate (~74.2%) compared to males (~18.9%).
* **Class:** Passengers in 1st Class were prioritized and had a higher chance of survival (~62.9%).
* **Age:** Children (0-18) showed the highest survival rate (~50.3%) among all age groups.
