# Titanic Dataset Analysis - Task 2 🚢

### Project Overview
This project is part of my **Data Science & Analysis Internship** at **Maincrafts Technology**. The goal of Task 2 is to perform Exploratory Data Analysis (EDA) on the famous Titanic dataset to identify factors that influenced the survival of passengers.

### Task Objectives
* **Data Loading:** Loaded the Titanic dataset containing passenger details like age, class, and gender.
* **Data Cleaning:** Handled missing values, specifically for the 'Age' column, using median imputation.
* **Exploratory Analysis:** Analyzed survival patterns based on Gender, Passenger Class, and Age Groups.
* **Data Visualization:** Created meaningful plots using Seaborn and Matplotlib to represent the findings.

### Key Insights
* **Gender:** Females had a significantly higher survival rate (~74.2%) compared to males (~18.9%).
* **Class:** Passengers in 1st Class were prioritized and had a higher chance of survival (~62.9%).
* **Age:** Children (0-18) showed the highest survival rate (~50.3%) among all age groups.

### Technologies Used
* **Language:** Python
* **Tools:** Google Colab
* **Libraries:** Pandas, Matplotlib, Seaborn
